# DRIFT paper — page thumbnails

Drop the paper's page images here. They are auto-discovered (no code edits) and
the thumbnail on `/papers` flips through them one page at a time on hover.

- Name them so they sort in reading order: `01.png`, `02.png`, `03.png`, …
  (zero-padded, since sorting is numeric-aware but padding is safest).
- Formats: png / jpg / jpeg / webp / avif.
- Portrait page scans look best; the card shows the full page (object-fit: contain).
- First file (`01.*`) is the cover shown before hover.
